# Shah Sultan’s IELTS Academy – Finance & Student Tracking

Production-ready multi-branch finance and student tracking system built with Next.js App Router, Prisma, PostgreSQL, and NextAuth Credentials.

## Quick Start

1. Copy `.env.example` to `.env` and update values.
2. Install deps: `npm install`
3. Generate Prisma client: `npx prisma generate`
4. Run migrations: `npx prisma migrate dev --name init`
5. Seed base data: `npm run seed`
6. Start dev server: `npm run dev`

Docs live in `docs/`.
